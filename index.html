PROJECT 01
Commercial KPI Dashboard
Pharma · Power BI

A pharma diagnostics company with 1,500+ active machines across clients had no unified view of commercial performance. Churn, delinquency, and operational KPIs were tracked in separate spreadsheets — updated manually, inconsistently, and always late.

Built a Power BI dashboard consolidating all KPIs into a single live view. Automated the data refresh pipeline via SQL + Python. Over 24 months, overdue balances tracked by the dashboard dropped ~28% — from ~BRL 250K to ~BRL 180K. Used by sales, operations, and C-level to make weekly decisions.

Power BI SQL Python DAX ETL
PROJECT 02
Automated Reporting Pipeline
Operations · Python · ETL

A reporting workflow required pulling data from multiple operational sources, applying business rules, and formatting results into a structured report. The process was entirely manual, took 2–3 days per cycle, and was prone to human error at every step.

Designed and built a Python ETL pipeline that automated extraction, transformation, validation, and output generation end-to-end. Processing time went from 2–3 days to under 10 minutes per run. Error rate dropped to near zero. The pipeline runs on schedule with no manual intervention required.

Python pandas SQL Scheduling Excel output
PROJECT 03
Data Migration Validation Framework
Pharma · Databricks · Data Quality

During an enterprise migration to Databricks, there was no systematic way to verify that data was moving correctly. Reports built on the new environment could be wrong — and no one would know until a business decision was affected. The risk was significant: executive dashboards and KPIs fed directly from this data.

Built a config-driven validation framework in Python that compares source vs. target datasets at row level. Generates structured HTML reports flagging discrepancies, null anomalies, and KPI inconsistencies. Exceptions are documented with business owner sign-off — distinguishing data problems from intentional migration decisions. Framework is reusable across migration phases.
